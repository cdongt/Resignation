## 安装方法





### ✅ 方法一：使用官方安装脚本并指定安装路径（推荐）

`uv` 官方提供了一个 PowerShell 安装脚本，支持通过环境变量 `UV_INSTALL_DIR` 指定安装目录。

#### 步骤如下：

1. **以管理员身份打开 PowerShell**（或确保你有写入目标路径的权限）。

2. **设置安装路径环境变量**（例如安装到 `D:\tools\uv`）：

   ```powershell
   $env:UV_INSTALL_DIR = "D:\tools\uv"
   ```

3. **运行官方安装命令**：

   ```powershell
   powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
   ```

4. **将安装路径添加到系统 PATH**（可选但推荐）：

   ```powershell
   [Environment]::SetEnvironmentVariable("Path", "$env:Path;$env:UV_INSTALL_DIR", "User")
   ```

   或者手动将 `D:\tools\uv` 添加到系统的环境变量 `PATH` 中。

5. **重启终端**，然后验证安装：

   ```powershell
   uv --version
   ```

---


### 🔄 更新 uv

如果使用方法一安装，后续可通过以下命令更新（同样可设置 `UV_INSTALL_DIR`）：

```powershell
$env:UV_INSTALL_DIR = "D:\tools\uv"
uv self update
```
