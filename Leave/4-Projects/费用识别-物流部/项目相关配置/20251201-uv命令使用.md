## 升级和维护依赖项

```
# 将所有项目依赖项升级到最新的兼容版本，并更新 lock 文件
uv update

```


## uv常用命令

### 创建、激活、退出虚拟环境

- 进入项目路径
- 创建虚拟环境
	- 不指定python版本

	  ```cmd
	  uv venv
	  ```
	  
	- 指定python版本

	  ```cmd
	  uv venv --python 3.11   # 指定 3.11 版本
	  ```
	  

- 激活虚拟环境

	```
	  .venv\Scripts\activate
	```
  
- 退出虚拟环境

	```
	  deactivate
	```


### 安装现有项目依赖项

```
# 从 pyproject.toml 安装依赖（推荐）
uv sync

# 如果你的项目使用 requirements.txt
uv pip install -r requirements.txt

```


### 依赖管理

- 安装依赖
	```
	  uv add requests
	```

- 卸载依赖
	```
	  uv remove requests
	```
- 升级依赖
	```
	  uv upgrade requests           # 升级单个依赖
	  uv upgrade --all              # 升级所有依赖
	```
- 导出依赖（生成 requirements.txt）
	```
	  uv export > requirements.txt
	```
  




### 运行项目或工具
```
python your_script.py

```

### 退出环境和清理

```
# 退出虚拟环境（返回系统环境）
deactivate

# 清理当前虚拟环境中的所有已安装包
uv pip uninstall -y -r <(pip freeze)
# 或者直接删除 .venv 文件夹
rm -rf .venv 

```
