## 步骤
### 步骤一：创建uv环境
> 可查看uv命令教程：[创建、激活、退出虚拟环境](20251201-uv命令使用.md#创建、激活、退出虚拟环境)]

```
uv venv --python 3.11
```

### 步骤二：安装mineru
```
uv pip install -U "mineru[core]"
```

### 步骤三：下载模型
```
mineru-models-download
```

### 步骤四：更改模型位置并修改mineru.json配置
> 以下是官网复制：[mineru官方说明文档](https://opendatalab.github.io/MinerU/usage/model_source/#switch-via-environment-variables)


- 下载完成后，模型路径将在当前终端窗口中输出，并自动写入`mineru.json`用户目录。
- [您也可以通过将配置文件](https://github.com/opendatalab/MinerU/blob/master/mineru.template.json)复制到您的用户目录并将其重命名为 来创建它`mineru.json`。
- 在本地下载模型后，您可以自由地将模型文件夹移动到其他位置，同时更新模型路径`mineru.json`。
- 如果将模型文件夹部署到另一台服务器，请确保将`mineru.json`文件移动到新设备的用户目录，并正确配置模型路径。
- 如果需要更新模型文件，可以`mineru-models-download`再次运行该命令。目前模型更新不支持自定义路径——如果您没有移动本地模型文件夹，模型文件将逐步更新；如果您移动了模型文件夹，模型文件将重新下载到默认位置并`mineru.json`进行更新。

### 步骤五：切换模型源
#### 当前窗口生效
```
set MINERU_MODEL_SOURCE=modelscope
```
- 仅在当前 CMD 窗口（及其子进程）中生效


#### 永久有效
```
set MINERU_MODEL_SOURCE=modelscope
```

- 永久有效
